Class NavigationStackFragment

  • All Implemented Interfaces:
    
public class NavigationStackFragment

    Manages a stack of different screens (fragments) that can be pushed and popped. This allows one screen to open a sub screen and then return back. Displays a bottom navigation bar which screens can customize and then listen to.
