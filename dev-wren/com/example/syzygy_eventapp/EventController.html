Package com.example.syzygy_eventapp

Class EventController

  • All Implemented Interfaces:
    
public class EventController

    Controller for reading/writing Invitation data in Firestore DB. EventViews will call this class to create, update, and observe events. Firestore is the source of truth; views have real-time listeners.
