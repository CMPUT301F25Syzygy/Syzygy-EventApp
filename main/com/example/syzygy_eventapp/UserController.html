Package com.example.syzygy_eventapp

Class UserController

  • All Implemented Interfaces:
    com.example.syzygy_eventapp.UserControllerInterface 
    
public class UserController
 implements UserControllerInterface

    Controller for reading/writing User data in Firestore DB.

    UserViews will call this class to create, update, and delete users and to get changes from the User model. Firestore has the true data, and snapshots are converted to User and delivered back to the UserView.
