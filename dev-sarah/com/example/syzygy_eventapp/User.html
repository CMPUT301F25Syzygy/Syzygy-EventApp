Class User

  • All Implemented Interfaces:
    
public class User

    User Model containing user profile data and active role. Fields are automatically updated when Firebase DB changes. Updates the Firebase DB when fields are set. Once the user is deleted it is still valid to read, but it will throw IllegalStateException when written to. Other classes like UserController and UserTest are responsible for initializing values upon creating a User.
